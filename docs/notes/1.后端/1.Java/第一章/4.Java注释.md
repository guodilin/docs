## 1. 代码注释

用于解释说明程序的文字

### 单行注释

```java
// 单行注释
```

### 多行注释

```java
/*
多行注释
*/
```

### 文档注释

```java
/**
文档注释
*/
```

### 注释演示

```java
/**
定义一个demo
控制台输出hello world
*/
public class Demo {
    /* 
    main方法程序的主入口
    */
    public static void main(String[] args) {
        // 控制台输出 hello world
        System.out.println("hello world");
    }
}
```
